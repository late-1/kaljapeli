# Kaljapeli

**Kaljapeli** (Beer Game) is a 2D game developed in **C#** using the **Jypeli** game library. Originally created as a coursework project for *Ohjelmointi 1*, it demonstrates object-oriented programming concepts, game physics, and event handling.

## Game Description

The goal of the game is to navigate the levels, collect items, and avoid obstacles. The game leverages the Jypeli engine's physics to handle collisions and movement.

**Key Features:**
* **Physics-based movement:** Smooth character control and gravity implementation.
* **Collectibles:** Score points by gathering items throughout the level.


## Controls

* **Arrow Keys / WASD:** Move character
* **Space:** Jump
* **Esc:** Pause / Exit

This project was created for educational purposes.
